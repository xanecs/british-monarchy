Algorithm of inheritance

Male Primogeniture

function MalePrimogeniture (monarch) {
  if (!monarch.hasChildren) {
    return MalePrimogeniture(monarch.previous);
  }

  if (monarch.hasSons) {
    return monarch.sons.eldest;
  }
  else if (monarch.hasDaughters) {
    return monarch.daughters.eldest;
  }
}

Algorithm of inheritance

Male Primogeniture

If the (now dead) Monarch has no children

Apply the rules to the previous Monarch

If the (now dead) Monarch has one or more sons

The eldest or only son becomes King

If the (now dead) Monarch has one or more daughters

The eldest daughter becomes Queen
